# Carpeta para el modelo entrenado en Teachable Machine

1. Ve a https://teachablemachine.withgoogle.com/train/image
2. Arrastra las carpetas de dataset_preparado/ (airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck)
3. Entrena el modelo
4. Exporta -> "Upload my model" (o descarga TensorFlow.js)
5. Coloca los archivos aquí: model.json, weights.bin, metadata.json
